# Benchmarking Multiconductor OPF taskforce
Respository for the math and data model spec of the Benchmarking Multiconductor OPF taskforce

This document curated and maintained by the IEEE PES Task Force on Benchmarking Multiconductor OPF for Distribution Systems and is designed to evaluate a well established version of the the Optimal Power Flow for Multiconductor distribution networks. 

A detailed description of this mathematical model is available in the root of this repository. Please launch issues to discuss issues with the report. Community-based recommendations and contributions are welcome and encouraged in all PGLib repositories. Please feel free to submit comments and questions in the [issue tracker](https://github.com/frederikgeth/bmopf-report/issues).  All data contributions are subject to a quality assurance review by the repository curator(s).

Specific versions of the report will be archived on arXiv.

## Citation Guidelines

This repository is not static.  Consequently, it is critically important to indicate the version number when referencing this repository in scholarly work.
